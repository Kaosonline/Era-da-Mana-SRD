### Numerical Resistance

**Escola** Abjuração; **Nível** Arcano 3, Divino 3

**CONJURAÇÃO**
**Tempo de Conjuração** 1 ação padrão  **Componentes** V, S

**EFEITO**
**Alcance** perto (7,5m + 1,5m/2 níveis)  **Alvos** ver texto  **Duração** ver texto  **Teste de Resistência** nenhum  **Resistência à Magia** não

**DESCRIÇÃO**
Numerical resistance permite que numerólogos usem o Caminho dos Números para conceder resiliência a seus aliados diante de perigos elementares. Role 5d8; este é seu vetor de dados. Você usará cada dado no vetor exatamente uma vez para personalizar numerical resistance. Escolha um dos dados e combine seu resultado com o valor apropriado na primeira coluna da Tabela das Oito Energias Básicas para determinar o tipo de energia ao qual numerical resistance concede resistência. Em seguida, aloque um dos dados para representar o número de criaturas que você pode afetar com numerical resistance (incluindo você mesmo). Todas essas criaturas devem estar dentro do alcance da magia. Então aloque um dos dados para representar o número de rodadas que numerical resistance dura. Finalmente, some os dois dados restantes. Este é o número de pontos de resistência que numerical resistance concede.
