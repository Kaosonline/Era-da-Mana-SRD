### Numerical Evocation

**Escola** Evocação; **Nível** Arcano 3

**CONJURAÇÃO**
**Tempo de Conjuração** 1 ação padrão  **Componentes** V, S

**EFEITO**
**Alcance** ver texto  **Alvos** ver texto  **Duração** instantânea  **Teste de Resistência** Reflexos reduz à metade  **Resistência à Magia** sim

**DESCRIÇÃO**
Você pode usar o Caminho dos Números para calcular as energias mágicas latentes ao seu redor e disparar raios personalizados de energia elementar que causam dano a alvos próximos. Role 1d6 por 2 níveis de conjurador que você possuir. Este é seu vetor de dados. Você usará cada dado no vetor exatamente uma vez para personalizar numerical evocation. Escolha um número de dados no vetor e combine seus resultados com os valores na primeira coluna da Tabela das Oito Energias Básicas para determinar os tipos de dano de energia que a magia causa. Você pode alocar qualquer número de dados dessa maneira, desde que ainda tenha dados suficientes restantes para as etapas subsequentes. Em seguida, aloque um dos dados para representar o número total de criaturas que você pode afetar, incluindo o alvo inicial. Não importa quantas criaturas você seja elegível para afetar, nenhuma das criaturas afetadas pode estar a mais de 4,5 m de distância uma da outra. Depois, aloque qualquer número dos dados para representar o alcance da resistência numerológica. O alcance da magia é 10 vezes a soma dos dados alocados. Finalmente, some os resultados dos dados restantes no vetor. Este é o valor do dano que numerical evocation causa à primeira criatura atingida. Um teste de Reflexos bem-sucedido reduz este dano pela metade. Alvos secundários recebem 50% do dano causado à primeira criatura atingida (arredondado para cima), e testes de Reflexos bem-sucedidos anulam este dano.
