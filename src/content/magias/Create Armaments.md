### Criar Armamentos

**Escola** Conjuração (Criação); **Nível** Arcano 3

**CONJURAÇÃO**
**Tempo de Conjuração** 1 minuto  **Componentes** V, S, M (diamantes com valor equivalente ao preço dos armamentos a serem criados)

**EFEITO**
**Alcance** 0 m  **Efeito** uma arma não-mágica, escudo ou conjunto de armadura  **Duração** instantânea  **Teste de Resistência** nenhum  **Resistência a Magia** não

**DESCRIÇÃO**
Você cria uma arma não-mágica, escudo ou conjunto de armadura. Para determinar se o item funciona como pretendido, você deve tentar um teste de Artesanato apropriado com a CD necessária para criar o item normalmente. Se você falhar, o item é criado com a condição de quebrado. Você pode criar itens feitos de materiais raros ou incomuns se incluir um pouco desse material como componente material adicional ao conjurar a magia, mas não pode criar itens de ferro frio com esta magia.
