### Influência Lendária

Pré-requisitos: Classe de poder espiritual menorOA.

Benefício:
Selecione imediatamente um talento (exceto um talento de criação de item) para cada espírito que você pode canalizar. Sempre que você realizar uma sessão espírita para canalizar um espírito, pode permitir que o espírito ganhe 1 ponto de influência sobre você para obter acesso ao talento de Influência Lendária daquele espírito enquanto canaliza aquele espírito. Você pode usar este talento bônus como pré-requisito para quaisquer talentos concedidos por um poder espiritual (como a habilidade campeão lendário do campeão), mas não para quaisquer outros talentos.

Descrição:
Cada lenda concede a você mais de seu poder em troca de maior influência.
