### Performance Introspectiva
**Tipo do talento**: Geral
**Pré-requisitos**: Executar (qualquer) 3 graduações, performance de bardo ou característica de classe de canção furiosa.
**Descrição**: Sem outros aliados para apreciar sua performance, você improvisa para torná-la mais inspiradora para si mesmo.
**Benefício**: Sempre que você for a única criatura afetada por uma performance de bardo ou canção furiosa que normalmente afeta todos os aliados dentro de um alcance, quaisquer bônus concedidos por sua performance de bardo ou canção furiosa são calculados como se seu nível de classe fosse 4 níveis mais alto. Este talento não permite que você ative uma performance de bardo ou canção furiosa se seu nível de classe real não for alto o suficiente para usar essa performance.
