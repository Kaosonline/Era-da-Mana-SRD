### Abençoar Equipamento Aprimorado

Pré-requisitos: Abençoar Equipamento, nível de conjurador 9°, classe de canalizar energia.

Benefício:
O número de usos de energia canalizada que você deve gastar para abençoar uma peça de equipamento usando Abençoar Equipamento e Alinhar Equipamento diminui em um (para um mínimo de um uso de canalizar energia por bênção). Você também pode abençoar uma peça de equipamento com até duas bênçãos ao mesmo tempo. Cada bênção fornece uma habilidade especial diferente da lista de Abençoar Equipamento (e da lista de Alinhar Equipamento se você possuir aquele talento). Você ainda deve ter um nível de conjurador alto o suficiente para abençoar uma peça de equipamento com cada uma das bênçãos selecionadas. Você gasta o número apropriado de usos de energia canalizada necessários para cada bênção, levando em conta a redução concedida por este talento (e o aumento para abençoar um item de qualidade superior, se necessário). Por exemplo, um clérigo com este talento precisaria gastar dois usos de energia canalizada para abençoar uma arma com as habilidades especiais matador de mortos-vivos e toque fantasma (um uso para matador mais um uso para toque fantasma). Se uma determinada peça de equipamento já tiver uma bênção, você pode dar a ela uma segunda bênção. Quando outra bênção é aplicada a uma peça de equipamento que atualmente tenha uma bênção ativa, as durações para as duas bênções são rastreadas separadamente.

Descrição:
Você pode canalizar ainda mais energia em armas e armaduras para conceder-lhes vários poderes divinos de uma vez.
