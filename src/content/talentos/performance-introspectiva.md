### Performance Introspectiva

Pré-requisitos: Atuação (qualquer) 3 graduações, recurso de classe performance de bardo ou canção feroz.

Benefício:
Sempre que você for a única criatura afetada por uma performance de bardo ou canção feroz que normalmente afeta todos os aliados dentro de um alcance, quaisquer bônus concedidos por sua performance de bardo ou canção feroz são calculados como se seu nível de classe fosse 4 níveis mais alto. Este talento não permite que você ative uma performance de bardo ou canção feroz se seu nível de classe real não for alto o suficiente para usar aquela performance.


Descrição:
Sem outros aliados para apreciar sua performance, você improvisa para torná-la mais inspiradora para si mesmo.
