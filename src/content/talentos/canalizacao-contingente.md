### Canalização Contingente

Pré-requisitos: Habilidade de classe curandeiro verdadeiro, Canalização Seletiva.

Benefício:
Você pode usar uma ação padrão para tocar um aliado e gastar um de seus usos diários de canalizar energia para criar um repositório de energia positiva dentro daquele aliado. Este repositório contém o mesmo número e tipo de dados que o normal para sua habilidade de canalizar energia, e dura por 1 minuto. Um aliado que tenha tal repositório pode usar uma ação imediata para rolar os dados do repositório e recuperar um número de pontos de vida igual ao resultado. Se um aliado que tenha tal repositório for reduzido a pontos de vida negativos, o repositório é ativado, permitindo que o aliado se cure sem usar uma ação.

Descrição:
Você pode imbuir outros com sua energia curativa para que possam usá-la no momento de maior necessidade.
