### Arremesso de Pegador de Touros

Pré-requisitos: Des 13, Estilo de Pegador de Touros, Agarrão Melhorado, Ataque Desarmado Melhorado; bônus base de ataque +8, brawlerACG nível 4º, ou monge nível 4º.

Benefício:
Quando você agarra com sucesso uma criatura com o talento Estilo de Pegador de Touros, pode imediatamente mover essa criatura para qualquer quadrado que você ameaça. Após ela ser posicionada, você pode quebrar o agarrar como uma ação imediata para mover a criatura mais 1,5 metros em qualquer direção. Você não pode usar esta manobra para mover um inimigo para um espaço intrinsecamente perigoso ou instável, como um poço, parede de fogo, ou no ar.


Descrição:
Você pode usar o ímpeto de um inimigo que investe para jogá-lo para o lado.
