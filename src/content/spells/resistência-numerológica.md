### Resistência Numerológica

**Escola** Abjuração; **Nível** Arcano 3, Divino 3

**CONJURAÇÃO**
**Tempo de Conjuração** 1 ação padrão  **Componentes** V, S

**EFEITO**
**Alcance** perto (7,5m + 1,5m/2 níveis)  **Alvos** ver texto  **Duração** ver texto  **Teste de Resistência** nenhum  **Resistência a Magia** não

**DESCRIÇÃO**
A resistência numerológica permite que numerologistas usem o Caminho dos Números para conceder resiliência a seus aliados frente a perigos elementais. Role 5d8; este é o seu pool de dados. Você usará cada dado no pool exatamente uma vez para personalizar a resistência numerológica. Escolha um dos dados e corresponda seu resultado ao valor apropriado na primeira coluna da tabela Oito Energias Básicas para determinar o tipo de energia ao qual a resistência numerológica concede resistência. Em seguida, aloque um dos dados para representar o número de criaturas que você pode afetar com a resistência numerológica (incluindo você mesmo). Todas essas criaturas devem estar dentro do alcance da magia. Depois, aloque um dos dados para representar o número de rodadas que a resistência numerológica dura. Finalmente, some os dois dados restantes. Este é o número de pontos de resistência que a resistência numerológica concede.
